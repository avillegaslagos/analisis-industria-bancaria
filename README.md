# Análisis de la Industria Bancaria Chilena con Datos CMF

## Descripción
Este proyecto desarrolla un prototipo en R para automatizar la obtención, limpieza y análisis de información financiera pública de la industria bancaria chilena, utilizando datos publicados por la Comisión para el Mercado Financiero (CMF).

El objetivo es reducir el desfase en la entrega de análisis comparativos entre instituciones financieras.

## Objetivo
- Automatizar la descarga de datos desde la CMF
- Limpiar y consolidar información financiera
- Generar indicadores comparables
- Visualizar variables clave

## Alcance
- Balances bancarios
- Dos instituciones
- Periodicidad mensual
- Análisis descriptivo

## Tecnologías
- R
- RStudio
- tidyverse
- GitHub
